# Cryptollama React Native App
React Native app for location based cryptollama game


###Set Up
####Start Metro
```shell script
npx react-native start
```

####Run ios app
```shell script
npx react-native run-ios
```

####Run Android app
```shell script
npx react-native run-android
```


###Resources

Libraries
